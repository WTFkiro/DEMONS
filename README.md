# DEMONS DRIVER 

## Python + OpenCV     
采集原视频中led信息，将结果格式化输出到output
该程序同样适用于其他8X8点阵视频，调用有亿点麻烦  
之后可能做成函数方便使用（可能）  

## Arduino + WS2812B   
基于FastLED库   
实现DEMONS驱动器的led面板效果      
持续更新中
